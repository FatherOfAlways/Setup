## Flags

URL: [`chrome://flags/`](chrome://flags/)

**Cross-platform:**

* `#camera-mic-preview` - Enabled
* `#one-time-permission` - Enabled
* `#enable-parallel-downloading` - speeds up downloads
* `#omnibox-drive-suggestions` - prioritizes text results over URL predictions in omnibox.
* `#extensions-menu-access-control` - Enabled - better extensions menu
* `#signature-based-sri` - Enabled - security stuff
* `#client-side-detection-brand-and-page-intent` - Enabled - security stuff
* `#client-side-detection-show-scam-verdict-warning` - Enabled - security stuff
* `#local-network-access-check` - Enabled (Blocking) - security stuff
* `#enable-canvas-noise` - Enabled - In all browsing modes - security stuff
* `#bind-cookies-to-scheme` - Enabled - security stuff

**Windows:**

* `#overlay-scrollbars`
* `#fluent-scrollbars`
* `#fluent-overlay-scrollbars`
