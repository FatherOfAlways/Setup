## Installation

1. Install dependencies:

	```
	sudo apt install coreutils sed git build-essential gcc-11 g++-11 lowdown
	```

2. Clone repository:

	```
	cd ~
	git clone https://github.com/aristocratos/btop.git
	cd btop
	```

3. Compile:

	```
	make
	```

4. Install:

	```
	sudo make install
	```
