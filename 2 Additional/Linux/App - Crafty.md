## Permission Setting

Replace *SERVER_ID* with actual folder name.

*Ownership:*

```
sudo chown crafty:crafty /var/opt/minecraft/crafty/crafty-4/servers/SERVER_ID -R
```

*Permissions:*

```
sudo chmod 2775 /var/opt/minecraft/crafty/crafty-4/servers/SERVER_ID -R
```
