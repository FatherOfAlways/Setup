## Nginx Proxy Manager Config

1. Navigate to Hosts > Proxy Hosts and click Add Proxy Host.

2. Configure the Details tab as follows:

	**Scheme:** https  
	**Forward Hostname / IP:** 172.16.0.1  
	**Forward Port:** 443  
	**Block Common Exploits:** Enable  
	**Websockets Support:** Enable
