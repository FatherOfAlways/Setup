## Usage

_homebrew-bundle comes preinstalled._

**Create**

Create empty Brewfile:

```
touch brewfile
```

**Dump**

You can create a Brewfile from all the existing Homebrew packages you have installed:

```
brew bundle dump
```

**Install**

On a new system, you can install all the software and its dependencies with:

```
brew bundle
```

or:

```
brew bundle --file </PATH/TO/FILE>
```
