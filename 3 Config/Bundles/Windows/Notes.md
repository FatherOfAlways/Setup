## Package Exceptions

Do not install the following via package managers:

- [**Brave**](https://brave.com/) - Updates and Sync not working.
- [**CapFrameX**](https://www.capframex.com/) - Only installs a CLI tool, not the GUI app.
- [**SpaceSniffer**](http://www.uderzo.it/main_products/space_sniffer/) - Only installs a CLI tool, not the GUI app.


Package glossary:

- **Guru3D.RTSS** = RivaTuner
